# progress-bar

Indicates the progress in a number bar format


## Description

We have all visited sites with a progress bar, whether it's to fill in information or for a survey. I have created this progress bar for you to use just for those special occasions.

[GitHub page](https://github.com/Digita1Panda/progress-bar)

[Progress Bar Page](https://digita1panda.github.io/progress-bar/)
